## Node Version

```bash
$ 20.10.0
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# watch mode (development)
$ npm run dev

# production mode
$ npm run build
$ npm run prod

# production mode (pm2)
$ npm run build
$ npm run pm2
```

## Running with Docker

```bash
$ docker-compose up
```
